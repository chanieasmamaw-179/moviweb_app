About MoviWeb

This application is a Flask-based web app called MoviWeb, designed for managing a personal movie collection. It utilizes SQLAlchemy for database interactions, specifically with a SQLite database, to store user and movie information.

Key Features:

  • User Management: Users can be added to the system, and each user can manage their own movie collection.
  • Movie Management: Users can add, update, and delete movies from their collection, with details like name, director, release year, and IMDb rating.
  • OMDb API Integration: The app fetches movie details from the OMDb API, allowing users to obtain up-to-date information when adding new movies.
  • Web Routes: The app provides various routes to interact with the user interface, including displaying users, their movies, and forms for adding or updating data.
  • Error Handling: It has basic error handling for 404 (not found) and 500 (server error) responses.
  • Database Testing: There's a route to test database connectivity and check the number of registered users.

Available Routes:

  • /users: Lists all users
  • /add_user: Adds a new user
  • /users/<int:user_id>/add_movie: Lists all movies for a specific user
  • /users/<int:user_id>/update_movie/<int:movie_id>: Updates an existing movie
  • /movies/<int:movie_id>/add_review: Adds a review for a movie
  • /reviews/<int:review_id>/update: Updates a review
  • /reviews/<int:review_id>/delete: Deletes a review
  • /add_director: Adds a new director
  • /directors: Lists all directors
  • /add_genre: Adds a new genre
  • /genres: Lists all genres
